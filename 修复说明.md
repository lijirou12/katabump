## 📊 技术原理

### 问题分析
旧版直接使用 CDP 发送点击事件，缺乏人类行为特征，容易被 Cloudflare 识别为自动化。

### 解决方案

```
1. 注入脚本 Hook attachShadow
   ↓
2. 多选择器查找 Turnstile checkbox
   ↓
3. 计算相对位置坐标
   ↓
4. 贝塞尔曲线鼠标移动（25 步）
   ↓
5. 悬停 150-350ms
   ↓
6. CDP 原生点击（80-180ms）
   ↓
7. 验证 "Success!" 标志
```

### 关键代码

```javascript
// 生成自然的鼠标移动路径
const path = generateBezierPath(startX, startY, clickX, clickY, 25);

// 沿路径移动鼠标
for (const point of path) {
    await client.send('Input.dispatchMouseEvent', {
        type: 'mouseMoved',
        x: point.x,
        y: point.y
    });
    await new Promise(r => setTimeout(r, 2 + Math.random() * 3));
}

// 悬停
await page.waitForTimeout(150 + Math.random() * 200);

// 点击
await client.send('Input.dispatchMouseEvent', {
    type: 'mousePressed',
    x: clickX,
    y: clickY,
    button: 'left',
    clickCount: 1
});
```
